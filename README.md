# QR Codes Generator

## Utilisation :

1. cloner de dépot :
```Bash
git clone https://github.com/scratch-2-0-2-4/QR-Codes.git
```
2. Aller dans le dossier :
```Bash
cd QR-Codes
```
4. Installer le paquet `qrcode` :

   **MacOS / Linux :**
   ```bash
   pip3 install qrcode[pil]
   ```

   **Windows**
   ```bash
   pip install qrcode[pil]
   ```

5. Lancer le programme :

   **MacOS / Linux :**
   ```bash
   python3 script.py
   ```

   **Windows :**
   ```bash
   python script.py
   ```

## TOTAL (toutes les commandes) :

**MacOS / Linux :**
```bash
git clone https://github.com/scratch-2-0-2-4/QR-Codes.git
cd QR-Codes
pip3 install qrcode[pil]
python3 script.py
```

**Windows :**
```bash
git clone https://github.com/scratch-2-0-2-4/QR-Codes.git
cd QR-Codes
pip install qrcode[pil]
python script.py
```
